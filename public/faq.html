Frequently Asked Questions

Everything you need to know about midpoint planning, structured data, pricing, and our SEO automation workflow.

What makes MeetSpot different from a basic midpoint calculator?

We combine centroid math, live venue data, and shareable SEO-ready pages. Your meeting recap already includes LocalBusiness, WebApplication, FAQPage, and Review schema so it can rank for keywords like “meeting location midpoint” or “group meeting point finder”.

How many people can I add?

The recommender supports 2-10 participants by default. If you need larger groups for events, clone the open-source repo, adjust MAX_PARTICIPANTS inside app/tool/meetspot_recommender.py, and redeploy.

Can I use Chinese addresses or keywords?

Yes. MeetSpot was born in Asia, so it fully supports 中文地址、聚会地点、会面点 等输入，并且仍会输出包含英文 SEO 关键词的页面，以兼顾本地化与全球搜索表现。

Is there an API?

Absolutely. Hit /api/find_meetspot with a JSON payload. We recommend using /docs for an interactive schema and /recommend for legacy compatibility.

How much does MeetSpot cost?

MeetSpot is free for now. We may launch premium collaboration features later, but the midpoint calculator and SEO toolkit will always have a generous free tier.

How do I keep SEO scores high?

Follow the SEO API Optimization Guide: ensure titles exceed 50 characters, meta descriptions ~155 characters, include structured data, and publish 500+ word sections. Track ranking deltas weekly via Keyword.com.

Can I embed the results elsewhere?

Yes. Each generated HTML file is static and can be uploaded to your CMS or referenced directly. You can also parse the output to create custom cards inside your product.

What if I need to remove a page from Google?

Update public/robots.txt to disallow the path or remove it from public/sitemap.xml, then request removal via Search Console’s URL tool.
